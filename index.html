Quando surgiram os primeiros computadores?

Voltar

Os primeiros computadores surgiram na década de 1940 o MARK 1 que era eletromecânico, surgiu em 1944 e o ENIAC que foi o primeiro computador eletrônico, em 1946, ambos criados nos EUA. Vale lembrar que os primeiros computadores eram máquinas gigantes que ocupavam uma sala inteira e pesava toneladas de quilos e que o funcionamento deles se davam por meio de válvulas e circuitos.

Evolução dos computadores

Voltar

A evolução dos computadores foi possível por conta da evolução dos componentes que estavam presentes no desenvolvimento dele. Após a invenção do transistor que eram peças pequenas que substituíram as válvulas, com isso foi possível diminuir bastante o custo e tamanho da fabricação dos computadores e isso se acentuou ainda mais com a criação dos circuitos integrados.

Computadores Pessoais

Voltar

Os PCs(Computadores pessoais) surgiram após a criação dos circuitos integrados, assim tornando viável o desenvolvimento de computadores pequenos para uso pessoal. Então foi em 1976 que surge o primeiro PC, o Apple 1, inventado por Steve Jobs.

O que é a Web?

Voltar

A World Wide Web, WWW ou apenas Web surgiu em 1991 e seu criador foi o Tim Berners-Lee. Ela é sistema presente na internet onde permite o acesso e conexão entre documentos de hypertexto, onde pode-se armazenar textos, imagens, vídeos e links, sendo que este último é responsável por ligar um documento de hypertexto a outro. Normalmente é através da Web e com o uso de um navegador que é possível buscar conteúdo na internet e acessar os coneteúdos presentes nela. Normalmente, usa-se a linguagem HTML(Hypertext Marketup Language) como linguagem para a criação de documentos hypertexto, esses arquivos tem a extensão ".html".

O que são navegadores?

Voltar

Os navegadores ou browsers são programas que permitem acessar a web, os sites e os arquivos HTML presentes nela. Através dos navegadores e com o auxílio dos mecanismos de busca como bing e google, é possível pesquisar pelos temas de interesse.

O que é a internet?

Voltar

A Internet é uma rede de computadores mundial que interliga e conecta todos os dispositivos entre si, possibilitando assim a troca e acesso de informações presentes em outro lugar ou dispositivo. A internet surgiu através do projeto da Arpanet, em meio a guerra fria, que tinha cunho militar e acadêmico. Certo tempo depois teve a união de várias redes de computadores interligando todos os continentes, assim, surgiu o que conhecemos hoje como "internet".

O que é client side?

Voltar

O client side ou lado do cliente é o lado no qual está presente os dispositivos que consomem os coneteúdos disponibilizados do lado servidor. No lado cliente temos presentes os desktops, smartphones e notebooks, no qual tem um navegador presente nele que acessa a internet e por consequência requisita os conteúdos presentes no lado servidor.

Linguagens client side

Voltar

Atualmente a linguagem mais utilizada no lado cliente para dar funcionalidade as páginas html é o javascript que é interpretado pelo subprograma V8 presente no navegador. Com isso, o navegador possibilita o uso de javascript no lado cliente.

O que é server side?

Voltar

O lado servidor no ambiente Web é onde está presente o servidor. O servidor é responsável por oferecer os serviços e atender as requisições por parte dos clientes, sendo que os servidores também armazenam as páginas html e o banco de dados, por exemplo. É onde fica armazenadas as informações que vamos acessar através do client side.

Linguagens server side

Voltar

As linguagens mais utilizadas atualmente no server side são: PHP, C#, Ruby, Java e Javascript. O Javascript é uma linguagem coringa que pode ser usada tanto no lado servidor quanto o lado cliente e isso é possível porque no server side há o uso do nodejs no servidor e este usa o subprograma V8 que permite o uso de javascript, assim utilizando tal linguagem de programação para funcionalidades no servidor.
